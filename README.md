# Misinformation Detection System

A simple web-based application to detect **misinformation** in user-provided text (and optional images) using a Flask backend.  
Designed for **academic projects, hackathons, and demos**.

---

## Problem Statement

Misinformation spreads rapidly through social media and digital platforms, leading to confusion and harmful decisions.  
This project helps users **identify misleading or sensational information** and classify it into meaningful categories.

---

## Features

- Text-based misinformation analysis  
- Optional image input support  
- REST API built with Flask  
- CORS enabled for frontend integration  
- Lightweight and fast backend  

---

## Technologies Used

- Python 3  
- Flask  
- Flask-CORS  
- HTML / CSS (optional frontend)
